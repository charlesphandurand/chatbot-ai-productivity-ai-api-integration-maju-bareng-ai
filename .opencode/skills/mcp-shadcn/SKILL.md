# MCP Skill: shadcn

Gunakan skill ini ketika pengguna meminta bantuan terkait `shadcn/ui` (komponen, pemasangan, contoh penggunaan, atau dark mode).

## Tooling / MCP yang tersedia
- MCP server: `shadcn` (lihat konfigurasi `mcp` di `opencode.json` dan/atau `.cursor/mcp.json`).

## Cara menjawab permintaan pengguna
1. Jika pertanyaan spesifik (mis. nama komponen, cara install, konfigurasi): gunakan MCP `shadcn` untuk mencari referensi dan ringkas poin pentingnya.
2. Jika pengguna butuh langkah praktis: berikan perintah/urutan langkah yang jelas (instalasi, konfigurasi, contoh snippet kode).
3. Jika tidak yakin: jelaskan asumsi, lalu minta klarifikasi data yang dibutuhkan (framework: Next.js/Vite/Astro/dll).

## Output format yang disarankan
- Ringkasan 3-6 poin.
- Bagian "Langkah" (bullet).
- Bagian "Contoh" (code snippet bila relevan).
